# README Generator
![License](https://img.shields.io/badge/license-The_MIT_License-blue.svg)

## Description

With special attention to license generation, this command-line application prompts users into inputting their desired README content and generates a markdown file based on that information. This application is designed to save programmersâ€™ precious time, so they can spend more time on coding. 

The project features node.js fundamentals.

To install necessary dependnecies, run the following command:

```

```
